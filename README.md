# IT Support Automation Toolkit

A console-based IT automation tool built with **C#** and **.NET** in **VS Code** to automate common troubleshooting and support tasks. This project was designed to simulate practical help desk and desktop support workflows while strengthening hands-on .NET development skills.

## Features

- Display system information such as machine name, operating system, processor count, and .NET version
- Check disk space and view available storage details
- Test network connectivity by pinging a host
- Show local IP address information
- Export a basic system report to a text file
- Provide a menu-driven interface for simple navigation

## Technologies Used

- C#
- .NET
- VS Code
- GitHub

## Project Purpose

This project was created to demonstrate practical experience with **.NET development** in an IT support context. Instead of building a generic application, this tool focuses on tasks that align with real-world desktop support and troubleshooting scenarios.

## How to Run

1. Clone the repository
2. Open the project in VS Code
3. Run the following command in the terminal:

```bash
dotnet run
```
